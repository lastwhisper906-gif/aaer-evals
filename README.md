# AAER Evals — 이해상충 없는 회계 품질 신호의 백테스트

> Authored by Claude Code, pending human audit (GA-001 (b), D15).
> 본 결과는 Claude 기반 단일 파이프라인에 한정된다 (PROJECT.md §5-5).
> **These controls BOUND memorization risk; they do not eliminate it.**

## 무엇인가

상장사 회계 품질에 대한 독립 신호는 구조적 공백이다: 감사인은 피감사인이
보수를 지급하고, 셀사이드는 거래 관계가 있으며, 신용평가사는 발행사가
수수료를 낸다. 이 저장소는 그 공백을 겨냥한 장치 — **LLM이 point-in-time
구조화 공시 데이터만으로 왜곡표시 위험을 스크리닝할 수 있는가** — 를 SEC
집행(AAER) 확정 사건 **8건** + 매칭 비집행 **대조군 22건**(사건당 2–3)으로
백테스트한 기록이다. 포지션 없음 · 교육·정보 목적 · 투자 조언 아님.

## 헤드라인 (발동 결론 규칙 = R3, 암기 전면 배치)

사전 커밋(`5f4ca65`, 대조군 점수 존재 전) 규칙의 기계 판정으로 **R3**이
발동했다. 헤드라인은 그 규칙이 강제하는 문장이다 (analysis/ISSUE_0_DRAFT.md §1):

> **이 케이스 집합에서 LLM은 확정 분식을 대조군과 분리한다 — 그러나 그 점수는
> 모델이 각 회사에 대해 *기억하는 것*과 얽혀 있다. 정체를 가린(교란) 채점도
> 여전히 분리하나(p = 0.0021, AUC 0.86), 이름-인지 프로브는 그 가림 자체가
> 불완전함을 보인다. 능력 질문은 컷오프 후 홀드아웃 — 암기가 구조적으로
> 불가능한 — 으로 넘어간다.**

**읽는 순서(교란/정체-가림 우선):** 점수는 회사 기억과 얽혀 있으므로 능력의
하한으로 anchor해야 할 값은 **정체-가림(교란) 프레임**이다.

- **정체-가림(교란) 실험군 vs 대조군**: 순열 p = **0.0021** / AUC **0.864**
  [0.722, 0.969] / 플래그 4/8. 정체를 지우고 모든 금액을 케이스 고정 계수로
  재스케일한 뒤에도 분리는 유지된다.
- **정체-노출(원본) 8 vs 22** (암기 기여 포함 — 얽힌 상한): 순열 p =
  **0.00114** (100k perms, one-sided), 평균차 **+19.8pp** (56.4 vs 36.5, 중위
  57.5 vs 33.0), AUC **0.824** [0.599, 0.983] — **N=30에서 불안정, 점 그림이
  1차 시각자료**, Cliff's δ 0.648, Fisher p = 0.0031 (플래그 임계 p≥50에서
  TP 6/8 · FP 3/22).
- **거짓양성**: 3/22 관측 ≈ 13.6% · **Clopper–Pearson 95% CI [2.9%, 34.9%]**.
  "0%"로 보고하지 않는다 (0 FP였어도 rule-of-three 상한 13.6%로 보고).
- **대조 기준선 (동일 30사·동일 PIT 데이터)**: Beneish M p=0.498/AUC 0.510 ·
  Dechow F p=0.268/AUC 0.573 — 정량 스크린은 이 표본에서 무분리. LLM 순위는
  둘과 사실상 무상관 (Spearman ρ = −0.075 / −0.144), 어느 쪽을 회귀로 빼도
  잔차 분리가 살아남는다 (p=0.0053 / 0.0010) — 사전 커밋 규칙 R2 **비발동**.
- **암기 분해 (R3 근거)**: 8케이스 중 **5건**이 사전 등록 R3 임계(정체 교란 Δ ≥
  케이스 분리 기여의 절반)를 넘었다 (Hertz −30.2pp = 최대, Monsanto는 유일하게
  인지 0 + 익명화 시 +15.8pp). 30사 익명 페이로드 이름 지목률 **50%**(대조군
  54.5%) — EDINET-Bench 표 데이터 <5% 대비 약 10배. 단 인지 자체는 대조군
  점수를 못 움직인다 (인지/비인지 대조군 중위차 +3pp < 사전 등록 10pp) — 동인은
  단순 친숙함이 아니라 각인된 스캔들 지식.
- **채점**: Claude 보조 + 인간 최종 확정 — 채점 26건 `human_finalized=true`
  (동결 `f3b76f7`). 채점자 claude-fable-5, 피평가자 **claude-sonnet-5(핀 고정,
  호출별 서빙 모델 검증, 핀 불일치 0 · INCOMPLETE 0)**.

발행 후보 초안(실회사 지명 공개 문서)은 **`analysis/ISSUE_0_DRAFT.md`** —
소유자 서명 대기, 이 저장소 상태에서는 미발행. 최종 게이트 패킷은

## 한계 (전문: docs/methodology_limitations.md)

L-1 모델 내부 지식은 차단 불가 — 측정·공개만 가능. L-2 실행층은 Claude Code
하네스 매개 (원시 API 아님). L-3 샘플링 파라미터 고정 불능 — 케이스 판정은
비결정론 표본의 점추정 (k=5로 정량화). L-4 격리는 실행별 게이트로 검증.
**L-5 교란은 수치 암기를 흩뜨릴 뿐 정체 인지를 제거하지 못한다**
("perturbation disrupts memorized NUMBERS, not IDENTITY recognition") — 이름
지목률 50%가 그 증거이며, 모든 양성 결과는 잔여 오염 하의 값이다. 선택·생존
편향: 실험군은 "적발까지 간 사건"의 생존 표본이고 대조군 라벨은 "무결"이 아니라
"비집행"이다 (Dyck-Morse-Zingales: 대형사 ~10% 연간 증권사기, ~⅓만 적발 —
측정 특이도는 하향 편향, 거짓양성 결과는 그만큼 보수적).

## 이것이 아닌 것

n=30(실험 8 + 대조 22)의 존재 증명 시도이지 성능 추정치가 아니다. 단일 분석자
(+AI)의 산출물이며 외부 재현·감사 전이다. 피평가자는 하네스 매개 호출이라
원시 API 재현과 다를 수 있다. 케이스 판정은 비결정론 표본이다. 정밀도/재현율 %
헤드라인은 의도적으로 없다 (n=8 신뢰구간 넓음, 실회사 벤치마크 비교 불가 —
사전 커밋 R4 프레이밍 제약). 특정 기업에 대한 주장이 아니며 — 현재 기업
산출물에는 "분식/fraud/조작" 서술을 쓰지 않는다 (§6) — 투자 조언은 더더욱
아니다.
